# Resume Bullets

## DS / ML Resume Version

- Built an end-to-end A/B testing analytics platform with FastAPI, PostgreSQL, Streamlit, Docker Compose, and pytest.
- Designed a normalized experiment data model for users, events, variants, assignments, metric definitions, and persisted statistical results.
- Implemented deterministic hash-based user assignment to control/treatment groups, making experiment analysis reproducible.
- Developed a metrics engine for conversion rate, ARPU, average order value, and purchase rate with uplift, confidence intervals, z-tests, and Welch's t-tests.
- Created a synthetic e-commerce event generator with realistic funnels, user activity segments, revenue events, and reproducible seeds.
- Added CI with GitHub Actions to run tests, compile checks, and Docker image build for portfolio-grade reproducibility.

## Product / Data Analyst Resume Version

- Created a product experiment lab that demonstrates the full lifecycle of A/B testing from event data to experiment results dashboard.
- Modeled product analytics entities including users, events, experiments, variants, assignments, metric definitions, and analysis outputs.
- Built an interactive Streamlit dashboard showing experiment status, group sizes, key product metrics, uplift, p-values, and confidence intervals.
- Implemented core product metrics such as conversion rate, revenue per user, average order value, and purchase rate for control/treatment comparison.
- Documented metric formulas, experiment lifecycle, data dictionary, limitations, and interview-ready project narrative.
- Packaged the project with Docker Compose, reproducible synthetic data, API documentation, tests, and GitHub Actions.
